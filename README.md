## 📌 About the Project

CivicConnect is a web-based platform that allows citizens to submit, track, and rate the resolution of civic grievances. Government officials can manage and resolve assigned complaints, while admins have full oversight with analytics and reporting.

---

## ✨ Features

- 🔐 Multi-role authentication — Citizen, Official, Admin
- 📝 Submit complaints with category, description, and location
- 📊 Real-time status tracking across 5 stages
- 🔧 Officials can update status and add remarks
- ⭐ Citizens can rate resolved complaints
- 📈 Admin analytics dashboard with resolution rates
- 🏙️ 6 complaint categories — Road, Water, Electricity, Waste, Safety, and more
